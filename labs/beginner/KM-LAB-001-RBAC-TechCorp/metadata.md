## Prérequis

Avant de réaliser ce laboratoire, il est recommandé de maîtriser :

* les Pods ;
* les Namespaces ;
* les objets Kubernetes ;
* les manifestes YAML ;
* les commandes de base `kubectl`.

---

## Compétences développées

À l'issue de ce laboratoire, les compétences suivantes sont acquises :

* Comprendre le fonctionnement de RBAC.
* Identifier les API Groups Kubernetes.
* Définir les verbes RBAC adaptés à un besoin métier.
* Créer un `Role`.
* Créer un `RoleBinding`.
* Associer un utilisateur à un rôle.
* Vérifier les permissions avec `kubectl auth can-i`.
* Tester les autorisations et les restrictions.
* Appliquer le principe du moindre privilège.
