# Kube Mentor Labs

Portfolio de laboratoires Kubernetes, DevOps et DevSecOps réalisés dans le cadre de ma montée en compétences.

L'objectif de ce dépôt est de regrouper des laboratoires pratiques, leurs solutions validées, les corrections, les bonnes pratiques et les retours d'expérience.

---

# Objectifs

* Approfondir Kubernetes par la pratique.
* Maîtriser les outils de l'écosystème Cloud Native.
* Construire un portfolio technique.
* Préparer les entretiens techniques DevOps/Kubernetes.
